# LibrarySystem

## 项目简介
LibrarySystem 是一个基于 Java 的图书馆藏品管理控制台程序，演示了面向对象设计中的**继承**、**多态**、**接口**以及**集合排序**等核心概念。系统支持：
- 管理两类藏品：**图书（Book）** 和 **DVD**。
- 借出 / 归还藏品，并检查借阅状态。
- 按标题对藏品进行排序（自然顺序）。
- 计算逾期罚金（不同藏品类型费率不同）。

## 类结构概览

Item (抽象类) Chargeable (接口)
├── Book       ├── getFine(int days)
├── DVD        └── (由 Book 和 DVD 实现)
│
Library   TestLibrary

- 管理 List<Item> - 测试用例
- 借阅/归还/显示/排序

| 类 / 接口     | 描述                                                         |
| ------------- | ------------------------------------------------------------ |
| `Item`        | 抽象基类，包含编号、标题、借出状态，实现 `Comparable` 按标题排序。提供 `borrow()` / `returnItem()` 等通用行为。 |
| `Book`        | 继承 `Item`，增加 `author` 属性，实现 `Chargeable`，罚金 0.5 元/天。 |
| `DVD`         | 继承 `Item`，增加 `duration`（分钟），实现 `Chargeable`，罚金 1.0 元/天。 |
| `Chargeable`  | 接口，定义 `getFine(int days)` 方法，用于计算逾期费用。      |
| `Library`     | 管理 `Item` 列表，提供增、查、借、还、显示（全部及排序）功能。 |
| `TestLibrary` | 主测试类，演示各类功能。                                     |

## 使用说明
### 运行环境
- JDK 8 及以上
- 命令行或任意 Java IDE

### 编译与运行
```bash
javac *.java
java TestLibrary
```

### 功能演示

程序将自动执行以下操作：

1. 初始化四件藏品（两本图书、两张 DVD）。
2. 显示所有藏品信息。
3. 按标题升序显示。
4. 尝试借出、重复借出、借出不存在的编号。
5. 尝试归还、重复归还、归还不存在的编号。
6. 演示罚金计算。

## 设计要点

- **多态**：`Library` 只依赖 `Item` 类型，可统一处理所有子类。
- **接口隔离**：`Chargeable` 单独定义计费行为，子类按需实现。
- **自然排序**：`Item` 实现 `Comparable`，基于 `title` 排序，符合业务需求。
- **封装**：借出状态由 `Item` 内部维护，外部通过 `borrow()` / `returnItem()` 改变。

## 注意事项

- 所有藏品编号应唯一（当前未做重复校验，可扩展）。
